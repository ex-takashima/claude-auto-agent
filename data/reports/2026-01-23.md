## LLM関連

### Google DeepMind

- **D4RT: Teaching AI to see the world in four dimensions**
  Google DeepMindは「D4RT（Dynamic 4D Reconstruction and Tracking）」という新しいAIモデルを発表しました。このモデルは、動画から3次元空間と時間の4次元における物体の動きを統一的に理解できます。D4RTは「ある画素がどこに、いつ、どのカメラから見えるのか」という質問に基づいた革新的なクエリシステムを採用しており、従来手法と比べ18倍から300倍高速で処理できます。1分間の動画をTPUチップで約5秒で処理可能で、物体追跡、3次元点群再構成、カメラ姿勢推定など、複数の4次元タスクに対応し、ロボティクスやAR、ワールドモデルなどへの応用が期待されています。

  🔗 https://deepmind.google/blog/d4rt-teaching-ai-to-see-the-world-in-four-dimensions/

### GitHub Blog

- **Build an agent into any app with the GitHub Copilot SDK**
  GitHubは、「GitHub Copilot SDK」を技術プレビューとして公開しました。このSDKは計画、ツール実行、ファイル編集、コマンド実行を可能にする実行プラットフォームで、Node.js、Python、Go、.NETに対応しています。開発者はGitHub Copilot CLIと同じエージェント機能を自分のアプリケーションに組み込むことができます。複雑なワークフロー構築を簡素化し、認証やモデル管理をGitHubが担当するため、アプリケーション開発に集中できます。既存のCopilot購読またはAPIキーを使用して利用可能で、YouTubeチャプタージェネレーターやカスタムGUIなど、すでに実用例が生まれています。

  🔗 https://github.blog/news-insights/company-news/build-an-agent-into-any-app-with-the-github-copilot-sdk/

## Claude Code

### Claude Code GitHub Releases

- **v2.1.17**
  Claude Code v2.1.17は、AVX命令セットに対応していないプロセッサでのクラッシュを修正した、安定性向上のためのアップデートです。従来の古いプロセッサやAVX命令に対応していない環境で発生していた問題を解決しました。このリリースは新機能を含まず、プロセッサ互換性の問題を解決することで、より多くの環境でclaude-codeが正常に動作することを実現しています。

  🔗 https://github.com/anthropics/claude-code/releases/tag/v2.1.17

- **v2.1.16**
  Claude Code v2.1.16では、新しいタスク管理システムが追加され、依存関係追跡などの新機能が実装されました。VSCodeにネイティブなプラグイン管理機能が追加され、OAuthユーザーがセッションダイアログからリモートClaudeセッションを参照・再開できるようになりました。また、サブエージェントの重い使用時にセッションを再開する際のメモリ不足クラッシュの修正や、`/compact`実行後の「残り文脈」警告表示の問題、セッションタイトルの言語設定の問題など、複数のバグ修正が行われました。ユーザー体験の向上とプラグイン管理の強化に焦点を当てた重要なアップデートとなっています。

  🔗 https://github.com/anthropics/claude-code/releases/tag/v2.1.16
