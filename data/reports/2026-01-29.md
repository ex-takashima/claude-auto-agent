## 技術ブログ

### GitHub Blog

- **From pixels to characters: The engineering behind GitHub Copilot CLI's animated ASCII banner**
  このGitHub Blogの記事は、Copilot CLIの3秒間のアニメーション化されたASCIIバナー開発の技術的過程を詳述しています。ターミナル環境ではキャンバスやレンダリングエンジンが存在しないため、"フレームバイフレームで手動再描画が必要"という制約下で、6,000行以上のTypeScriptコードが必要となりました。ANSI色コードはターミナル間で不一致が生じるため、チームは"セマンティックな役割"(目、ゴーグル、影など)をANSI色にマッピングし、異なるテーマやアクセシビリティ設定に対応させました。スクリーンリーダー対応、高コントラストモード、色覚障害対応など、CLIのアクセシビリティ設計はWebより研究が進んでいないため、ゼロから構築が必要でした。この事例は、ターミナルUI開発が表面的には単純に見えても、実現には相当の工学的課題と革新的なツール開発が必要であることを示しています。

  🔗 https://github.blog/engineering/from-pixels-to-characters-the-engineering-behind-github-copilot-clis-animated-ascii-banner/

- **Year recap and future goals for the GitHub Innovation Graph**
  GitHubは2026年1月に、Innovation Graphの第2回目の定期データ更新を発表しました。このデータセットは「公開されたソフトウェア開発活動に関する集計統計の安定した定期的な情報源」として機能し、政策立案や研究、資金配分の決定を支援しています。過去1年間、Innovation Graphのデータは学術論文や国際報告書、主要メディアで広く活用され、アフリカのデジタル能力やオープンソース協力構造、国家レベルのソフトウェア経済複雑性など、多角的な分析に貢献しています。

  🔗 https://github.blog/news-insights/policy-news-and-insights/year-recap-and-future-goals-for-the-github-innovation-graph/

## Claude Code

### Claude Code GitHub Releases

- **v2.1.22**
  Claude Code v2.1.22は2026年1月28日にリリースされました。このバージョンの主な変更点は、非インタラクティブモード（-pオプション）での構造化出力に関するバグ修正です。ユーザーが構造化出力を使用する際に、-pフラグを指定した場合の処理が正しく機能するように改善されました。このリリースにはアセット2つが含まれており、コミットハッシュは`73eddfd`です。

  🔗 https://github.com/anthropics/claude-code/releases/tag/v2.1.22

- **v2.1.21**
  Claude Code v2.1.21は、2025年1月28日にリリースされた更新版です。日本語IMEの全幅（ぜんはば）数字入力に対応し、オプション選択プロンプトでの入力が改善されました。VS Code拡張機能に自動Python仮想環境アクティベーション機能が追加されました（`claudeCode.usePythonEnvironment`設定で制御可能）。ファイル操作ツール（Read、Edit、Write）を優先するようにClaudeの動作が改善されました。バグフィックスとして、シェル完了キャッシュファイルが終了時に切り詰められる問題を修正、ツール実行中に中断されたセッション再開時のAPIエラーを修正、Windows上でのVS Code拡張機能ファイル検索の動作不具合を解決しました。

  🔗 https://github.com/anthropics/claude-code/releases/tag/v2.1.21
