## Claude Code

### Claude Code GitHub Releases

- **v2.1.15**
  Claude Code v2.1.15では、npmインストールの廃止予定通知が追加され、ユーザーは`claude install`コマンドの実行またはドキュメントの確認が推奨されています。React Compilerを活用したUIレンダリング性能の向上が実施されました。`/compact`コマンド実行後に「Context left until auto-compact」警告が消えない問題が修正されました。また、MCP stdioサーバーのタイムアウト処理が改善され、子プロセスが正常に終了されるようになり、UIフリーズの問題が解決されました。

  🔗 https://github.com/anthropics/claude-code/releases/tag/v2.1.15

- **v2.1.14**
  Claude Code v2.1.14では、bashモードで履歴ベースのオートコンプリート機能（`!`キー）が追加され、部分的なコマンドを入力してTabキーを押すことで過去のコマンド履歴から補完できるようになりました。また、インストール済みプラグインリストに検索機能が追加され、プラグインを特定のgitコミットSHAにピン止めできるようになりました。コンテキストウィンドウの制限計算が改善され、約65%ではなく意図した98%での制限に修正されました。並列サブエージェント実行時のメモリ問題とシェルコマンド後のストリームリソースメモリリークが解決されました。

  🔗 https://github.com/anthropics/claude-code/releases/tag/v2.1.14
